# 幾何パターンランダム生成プロトタイプ

## 概要

次の幾何パターンをランダムに生成するアルゴリズムのプロトタイプを作成する。

- バウハウス・パターン
- ジオメトリック・パターン

## 技術スタック

- パターンごとに`.ts`ファイルを作成し、TypeScriptで実装する
- `tsx`コマンドで実行することで、SVGファイルが生成されるようにする
- SVGの組み立て処理にはSVG.jsを使用する

## 実装要件

- 正方形のSVG画像として生成する
- 生成するパターンのサイズは300px四方程度を想定する
- 各パターンの生成アルゴリズムは、3色のHEXカラーコードを入力とする関数としてまとめる
- 3色のうち、インデックスが小さいほど色の使用面積が大きくなるようにする
- 生成したSVGコードは、`.generated/{patternName}/{YYYYMMDD-THHMMSSZ}.svg`ファイルとして出力する
  - 例：`.generated/bauhaus/20240601-T120000Z.svg`
- コンソールには生成したSVGファイルへのパスと、生成にかかった時間を出力する

## 各パターンの詳細

### バウハウス・パターン

円・矩形・アーチ・直線などの幾何学的図形をグリッド上にランダム配置する。

大まかな手順：
1. 正方形グリッドを作る
2. 各セルに四角、円、横線、縦線、半円アーチのどれかを置く

コツ：
- 回転は 0 / 90 / 180 / 270 度 に限定する
- 色は1セルにつき1色、背景は固定
- 1セルに置く図形は基本1個、多くても2個まで
- たまに2x2セルをまとめて大きい図形にする

### ジオメトリック・パターン

三角形を基本とし、以下のようなバリエーションを考える。

想定するパターン：
- 同じ形状・サイズの三角形を斜めにランダム回転させて敷き詰めたパターン
- 濃淡と形状の異なる三角形をタイリングしたパターン
- さまざまな形状の三角形をランダム配置し、重なり合う部分の色をブレンドするパターン
