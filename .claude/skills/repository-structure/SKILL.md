# ドキュメント構造定義スキル

このスキルは、リポジトリ構造定義書に最新のディレクトリ構成を反映するためのガイドラインです。

## 更新対象

次のファイルを更新対象とします。

- `docs/repository-structure.md` - リポジトリ構造定義書

## 更新手順

現在のディレクトリ構造を確認し、次のように更新を行ってください。

1. ルート**直下**のファイルやディレクトリの一覧を`## 全体構造`に反映する
   - ディレクトリ内の再帰的な検索は行わない
2. `app/`配下のファイルやディレクトリの一覧を`## appディレクトリ`に反映する
   - ディレクトリ内のファイルやディレクトリも再帰的に検索し、一覧に記載する
3. `docs/` **直下**のファイルやディレクトリの一覧を`## docsディレクトリ`に反映する
   - ディレクトリ内の再帰的な検索は行わない
4. `proto/` **直下**のファイルやディレクトリの一覧を`## protoディレクトリ`に反映する
   - ディレクトリ内の再帰的な検索は行わない
