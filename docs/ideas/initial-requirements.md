# Webアプリ「PCCS Lens」の初期アイデア

## アプリのコンセプト

> 「世界の色をPCCSというレンズを通して見る」

PCCS（Practical Color Co-ordinate System：日本色研配色体系）は、日本色彩研究所が開発した、色彩調和を目的としたカラーシステムです。色相（12色相）と、明度・彩度を融合した「トーン（12区分）」の2つの要素で色を体系化し、イメージに合わせた配色や色選びを容易にします。日本の色彩検定でも、PCCSをベースに色や配色を考える問題が多く出題されます。

このアプリでは、さまざまな色をPCCSで捉え、PCCSの枠組みに落とし込んで配色を考える上で便利なツールを複数提供します。

## ページごとの機能の構想

1ページに1つの機能を実装します。現段階で計画している機能は2つです。

### 機能1. 色をPCCS値・JIS慣用色に近似

ユーザーがUIにカラーコードを入力したら、次の処理を行います。

- 入力されたカラーコードに近い色のPCCS値を上位8件表示する
- 入力されたカラーコードに近い色のJIS慣用色名を上位4件表示する

ユーザーが入力したカラーコード、近似値となる8つのPCCS値、4つのJIS慣用色名には、それぞれの色プレビューも添えて表示します。

ユーザーは、オプション選択UIで、検索対象となるPCCS値やJIS慣用色を絞り込むことができます。

- 検索対象となるPCCS値のフィルタリングオプション
  - 「新配色カード199にある色」：`data/pccs_colors.csv`から検索
  - 「すべての色」：`data/pccs_colors_full.csv`から検索
- 検索対象となるJIS慣用色のフィルタリングオプション
  - 「色彩検定3級に出題される色」：`data/jis_colors_aft_grade-3.csv`から検索
  - 「色彩検定2~3級に出題される色」：`data/jis_colors_aft.csv`から検索
  - 「すべてのJIS慣用色名」：`data/jis_colors.csv`から検索

### 機能2. 配色の分析とインタラクティブな調整

ユーザーは複数色のカラーコードを入力することができます。
入力されたそれぞれの色について、最も色をよく近似できるPCCS値を推定し、その値に基づいて次のような表示を行います。

- 入力された各色を近似するPCCS値とその色プレビューを表示する
- PCCS色相環を描画し、色相環上で、先ほど推定したPCCS近似値の色相をハイライト表示する
- PCCSトーン概念図を描画し、トーン概念図上で、先ほど推定したPCCS近似値のトーンをハイライト表示する

ここまでで、ユーザーは自分の入力した色のPCCS色相やトーンの関係を理解することができます。

さらに、「配色の分析」クロージャを開くと、次の分析結果が表示されます。
各項目ごとに、当てはまる「」内のラベルを表示します。

- 色相同士の関係（※入力色が2色の場合のみ表示）
  - 「同一色相」：色相差が0の場合
  - 「隣接色相」：色相差が1の場合
  - 「類似色相」：色相差が2~3の場合
  - 「中差色相」：色相差が4~7の場合
  - 「対照色相」：色相差が8~10の場合
  - 「補色色相」：色相差が11~12の場合
- トーン同士の関係（※入力色が2色の場合のみ表示）
  - 「同一トーン」：トーンが同じ場合
  - 「類似トーン」：トーン概念図上でトーンが隣り合っている場合
  - 「対照トーン」：上の2つに当てはまらない場合
- ハーモニー（※入力色が2色の場合のみ表示）
  - 「ナチュラルハーモニー」：色相が20:Vに近い色より、色相が8:Yに近い色の方が、トーン概念図上で上に位置する場合
  - 「コンプレックスハーモニー」：色相が8:Yに近い色より、色相が20:Vに近い色の方が、トーン概念図上で上に位置する場合
- 色相・トーンを軸とした配色技法
  - 「ドミナントカラー」
  - 「ドミナントトーン（トーンイントーン）」
  - 「トーンオントーン」
  - 「トーナル」
  - 「カマイユ」
  - 「フォカマイユ」
  - 「ビコロール」
  - 「トリコロール」
- 色相環の分割による配色技法
  - 「ダイアード」
  - 「トライアド」
  - 「スプリットコンプリメンタリー」
  - 「テトラード」
  - 「ペンタード」
  - 「ヘクサード」

ユーザーは、PCCS色相環やトーン概念図上のハイライトされた箇所を操作することで、色相やトーンを調整することができます。
この操作により、配色のプレビューや配色の分析もインタラクティブに更新されます。
