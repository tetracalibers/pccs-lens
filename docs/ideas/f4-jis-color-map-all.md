# F4：慣用色名比較マップ（全色）

## 実装上の留意事項

- コンポーネント固有でないロジックやユーティリティは @app/src/lib/jis-color-map 配下にTSファイルとして切り出す
  - ただし、マンセル値関連のロジックは @app/src/lib/color/munsell.ts に実装する
  - また、データ取得に関するロジックは @app/src/lib/data/jis-colors.ts に実装する
- @app/src/lib/color 配下と @app/src/lib/data 配下の関数を積極的に利用する
- 今回新規実装するページで使うsvelteコンポーネントは @app/src/lib/components/jis-color-map 配下に配置する
- このメモの中に書かれたコードはすべて疑似コードであり、変数名、プロパティ名、Props名などは仮称なので、適切な命名を考えて実装する

## コンポーネント

### 新規作成：慣用色名マップ

指定されたfamily（色みのグループ）に属する慣用色を色相・明度順に並べたマップを描画する。

```ts
interface Props {
  groupId: ColorFamily | ColorSubfamily | "all"
}
```

- 画面幅が狭くはみ出す場合は、横スクロール可能にする

#### 色相範囲の算出

描画対象となる慣用色が属する全subfamilyの`hintPCCSHue`を調べ、`from`の最小値と`to`の最大値を算出する。

最小`from`と最大`to`の範囲内のvトーンの色（奇数色相を含む）は後ほどセルとして表示するため、色オブジェクトの配列を保持しておく。以降、この配列を`helpPCCSList`と呼ぶ。

#### 明度範囲の算出

描画対象となる全慣用色と`helpPCCSList`内全色の`munsell`プロパティを`parseMunsell`関数でパースし、含まれる最大明度と最小明度を求める。

- 明度は`2.5`から`9.0`までの`0.5`刻みの数値として扱う
- 明度`5.0`が必ず含まれるように最大明度と最小明度を決定する
- 同明度の色が複数ある場合は、それぞれのマンセル彩度を調べ、現れた彩度の個数をカウントする

#### 明度スケールの描画

- 上に行くほど明度が大きくなるように、含まれる最小明度〜最大明度の範囲内の明度スウォッチを縦に並べる
- 同明度の色が複数ある場合は、先ほど算出した彩度の個数を明度スウォッチにPropsとして渡す

#### 等明度軸の描画

- 明度5.0の高さには、色相の個数分だけ明度5.0の明度スウォッチを横に並べる

#### セルの配置決定

- 描画対象となる全慣用色と`helpPCCSList`内全色のマンセル色相を、`10RP`を最も若い値としてソートする
- 横方向には色相順に、縦方向には明度順にセルを配置する
- 同明度の色が複数ある場合は、マンセル彩度の低いものほど上に来るようにセルを配置する。この時、同彩度の色が同じ縦位置に並ぶようにする
- 明度5.0の色は、等明度軸上の明度5.0の明度スウォッチを置き換える形で配置する
- 無彩色は、明度スケール上の明度スウォッチを置き換える形で配置する

#### セルの描画

- 慣用色名は慣用色スウォッチ、PCCS色相はPCCS色スウォッチで表示する
- 同じマンセル値の慣用色が複数ある場合は、慣用色スウォッチに複数の慣用色名を配列で渡す
- 慣用色と参考PCCS色が同じマンセル値の場合は、慣用色スウォッチにPCCS色記号も渡す

### 新規作成：明度スウォッチ

慣用色名マップで、明度を表すセルとして使う。
同明度が複数ある場合にも使えるよう、セル何個分の高さを確保するかを指定できる。

```ts
interface Props {
  // 明度を表す数値
  value: number
  // セル何個分の高さを確保するか
  // デフォルトは`1`
  scale?: number
}
```

- 正方形を`scale`倍だけ縦に引き伸ばした長方形
- 角丸、塗りなし
- 枠線と文字の色は`var(--color-body)`
- 中央に`明度${X.X}`のラベルを表示

### 新規作成：慣用色スウォッチ

慣用色名マップで、慣用色を表すセルとして使う。

- 角丸の正方形を、その慣用色のHEXコードで塗りつぶす
- 中央に慣用色名を表示する
- `nameSegments`が指定されている場合は、その要素ごとに改行して表示する
- 複数の慣用色名がPropsで与えられた場合は、縦に並べて表示する
- PCCS記号もPropsで与えられた場合は、正方形の上にそのPCCS記号を表示する
- 正方形内に収まるように、文字数に応じて文字サイズを調整する
- 文字色は`isLightColor`関数によって見やすい色に分岐させる

### 新規作成：PCCS色スウォッチ

慣用色名マップで、PCCS色相を表すセルとして使う。

- 円をそのPCCS色相のvトーンのHEXコードで塗りつぶす
- 中央に色相記号（`${色相番号}:${色相記号}`）を表示する
- 円内に収まるように、文字数に応じて文字サイズを調整する
- 文字色は`isLightColor`関数によって見やすい色に分岐させる
