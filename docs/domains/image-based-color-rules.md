# イメージ別配色ルール定義書

本ドキュメントは、PCCS Lens F3「イメージ別配色シミュレータ」において各イメージ（テーマ）の配色サジェストを実装するためのルール定義書。
`docs/domains/color-analysis-rules.md` および `docs/domains/glossary.md` の用語・定義を前提とする。

---

## 1. 共通定義

### 1-1. 用語定義

| 用語 | 定義 |
|---|---|
| 純色 | vトーン（最高彩度） |
| 明清色 | p, lt, bトーン |
| 心理四原色 | 色相番号 2:R, 8:Y, 12:G, 18:B |
| ブラウン系色相 | 色相番号 1〜10（pR〜YG） |
| パープル系色相 | 色相番号 19〜24, 1（pB〜RP〜pR） |
| BG〜B系色相 | 色相番号 14〜19（BG〜pB） |

### 1-2. トーン彩度分類

実装上のトーン彩度レベル分類：

| 彩度レベル | トーン記号 | トーン概念図の列 |
|---|---|---|
| 最高彩度 | v | 1列目 |
| 高彩度 | b, s, dp | 2列目 |
| 中彩度 | lt, sf, d, dk | 3列目 |
| 低彩度 | p, ltg, g, dkg | 4列目 |
| 無彩色 | W, ltGy, mGy, dkGy, Bk | — |

### 1-3. 「1段階鮮やかなトーン」の定義

トーン概念図で1列右（低彩度→中彩度、または中彩度→高彩度）に移動したトーン。
同一の行（明度帯）を保つ。

| 元のトーン | 1段階鮮やか（= 1列右） |
|---|---|
| p | lt |
| ltg | sf |
| g | d |
| dkg | dk |
| lt | b |
| sf | dp |
| d | dp |
| dk | dp |

> 無彩色（W, ltGy, mGy, dkGy, Bk）は彩度列に属さないため、この定義の適用外とする。

### 1-4. 「高彩度」「強調色」の定義

**高彩度トーン（強調色）**: v, b, s, dp

「強調色はNG」のテーマでは、アクセントカラーのサジェスト対象から v, b, s, dp を除外する。

### 1-5. 「アソートカラーと区別する必要はない」の定義

アクセントカラーのサジェスト条件がアソートカラーのサジェスト条件と完全に同一であることを意味する。
UIの表示は他テーマと同様（「アクセントカラーを追加する」ボタンあり）だが、サジェスト計算ロジックをアソートと同一にする。

### 1-6. 面積比率

| 状態 | ベース : アソート : アクセント |
|---|---|
| アクセントカラーあり（通常テーマ） | 6 : 3 : 1 |
| アクセントカラーなし | 7 : 3 |
| ダイナミックテーマ（常に3色均等） | 1 : 1 : 1 |

### 1-7. サジェストの共通仕様

1. サジェストは「推奨の色相・トーンのハイライト」であり、選択の制限ではない
2. サジェスト対象外の色相・トーンも選択可能だが、UIでは目立たない表示とする
3. 初期値はサジェスト内からランダムに選択する
4. 初期値の決定順序：ベースカラー → アソートカラー → アクセントカラーの順に連鎖的に決定する
5. サジェスト対象が0件の場合：テーマのトーン範囲・色相範囲全体をサジェスト対象とする（ハイライトなし状態）

### 1-8. アクセントカラーUIの共通仕様

- 全テーマで「アクセントカラーを追加する」ボタンを表示する
- 初期表示は2色（ベースカラー・アソートカラー）
- アクセントカラーUIが非表示の間は、配色プレビューや幾何パターンにアクセントカラーを含めない
- ボタン押下でアクセントカラーの入力UIを追加表示する
- ラベルは全テーマ統一：**ベースカラー / アソートカラー / アクセントカラー**

### 1-9. 無彩色選択時のHueWheelの挙動

トーン選択UIで無彩色（W, ltGy, mGy, dkGy, Bk）が選択されている場合：

- 色相を持たないため、HueWheelの「選択中の色相ハイライト」は表示しない
- すべての扇形をデフォルト状態で表示する
- ユーザーが色相扇形をクリックした場合は「有彩色への切り替え」として処理し、トーン選択UIをそのテーマで有彩色として使えるサジェストトーンからランダムに選んで更新する

## 2. テーマ別定義

**定義の見方：**

- **色相範囲** (`allowedHues`): テーマ全体で選択可能な色相番号の集合
- **トーン範囲** (`allowedTones`): テーマ全体で選択可能なトーン記号の集合
- **静的サジェスト**: 他の役割の選択値によらない固定のサジェスト条件
- **動的サジェスト**: 他の役割の選択値に基づいて計算するサジェスト条件

色相差の計算は `color-analysis-rules.md §6` の定義（24色相環上の最短距離）を用いる。

---

### 2-1. エレガント (elegant)

**イメージ**: 女性的な・気品のある・洗練された・優雅な
**配色の特徴**: パープル系の明清色・高明度中間色の組み合わせ。明度差を抑えた上品なコントラスト。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 19, 20, 21, 22, 23, 24, 1（pB〜pR のパープル系7色相） |
| トーン範囲 | p, ltg, lt, b |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 19〜24, 1（テーマ色相範囲全体） |
| トーン | p, ltg |

#### アソートカラー（動的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 19〜24, 1（テーマ色相範囲全体） |
| トーン | ベースカラーの「1段階鮮やかなトーン」を参照（下表） |

**ベースカラーのトーン→アソートカラーのサジェストトーン対応表：**

| ベースのトーン | アソートのサジェストトーン |
|---|---|
| p | lt |
| ltg | lt |

> `ltg` の「1段階鮮やか」は `sf` だが、エレガントは明清色（p, ltg, lt, b）のみで構成する配色のため、sf はいかなる場合もサジェストに含めない。`ltg` ベースの場合も `lt` をサジェストする。

#### アクセントカラー（静的サジェスト）

「強調色はNG」= v, b, dp を除外

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 19〜24, 1（テーマ色相範囲全体） |
| トーン | p, ltg, lt（b, dp, v は除外） |

---

### 2-2. カジュアル (casual)

**イメージ**: 明るい・活発な・親しみやすい
**配色の特徴**: 純色から明清色の範囲で、色相対比のある明るい配色。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 1〜18（pR〜B の全18色相） |
| トーン範囲 | p, lt, b, s, v |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 3, 4, 5, 6, 7, 8（rO〜Y の橙〜黄系6色相） |
| トーン | p, lt |

#### アソートカラー（動的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | ベースカラーとの色相差が **4以上**（中差〜補色）かつ 1〜18の範囲内 |
| トーン | p, lt, b, s, v（テーマ範囲全体） |

> 色相差 0〜3 の色相はサジェスト対象外（選択は可能）。

#### アクセントカラー（動的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | ベースまたはアソートとの色相差が **4以上** かつ 1〜18の範囲内 |
| トーン | b, s, v（高彩度色） |

---

### 2-3. クラシック (classic)

**イメージ**: 重厚な・円熟した・伝統的な
**配色の特徴**: 暗く落ち着いた色調。ブラウン系に別の色相を添える演出。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 1〜18（pR〜B の全18色相） |
| トーン範囲 | g, dkg, dk, dp, Bk, dkGy |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 1〜10（pR〜YG のブラウン系10色相） |
| トーン | dkg, dk |

#### アソートカラー（動的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | ベースカラーとの色相差が **0〜3**（同一〜類似色相）かつ 1〜18の範囲内 |
| トーン | g, dk |

#### アクセントカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 11〜18（yG〜B の低明度の緑・青系） |
| トーン | dp, Bk, dkGy |

---

### 2-4. クリア (clear)

**イメージ**: 明るい・爽やかな・透明感のある
**配色の特徴**: BG〜B系寒色の明清色とホワイト系の組み合わせ。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 14, 15, 16, 17, 18, 19（BG〜pB の6色相） |
| トーン範囲 | p, lt, W, ltGy |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 14〜19（テーマ色相範囲全体） |
| トーン | p, lt, W, ltGy（テーマ範囲全体） |

#### アソートカラー（動的サジェスト）

| ベースのトーン種別 | アソートのサジェストトーン |
|---|---|
| 無彩色（W, ltGy） | p, lt |
| 有彩色（p, lt） | W, ltGy |

色相: 14〜19（テーマ色相範囲全体）

#### アクセントカラー（静的サジェスト）

「強調色はNG」= v, b, dp を除外

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 14〜19（テーマ色相範囲全体） |
| トーン | p, lt, W, ltGy |

---

### 2-5. シック (chic)

**イメージ**: 渋い・洗練された・大人っぽい
**配色の特徴**: やや色みを感じさせる無彩色系の組み合わせ。高彩度色は含まない。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | **20〜24, 1〜3** または **8〜16**（暖色の中間色相 4〜7 を除外することで奇抜な印象を避ける。また、17〜19はモダンな印象に近づくため除外） |
| トーン範囲 | dkg, dk, g, ltg, Bk, dkGy, mGy |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | テーマ色相範囲全体 |
| トーン | Bk, dkGy, mGy, g（無彩色系または g トーン） |

#### アソートカラー（動的サジェスト）

| ベースのトーン種別 | アソートのサジェストトーン |
|---|---|
| 無彩色（Bk, dkGy, mGy） | dkg, dk, ltg, g（有彩色系） |
| 有彩色（g） | Bk, dkGy, mGy（無彩色系） |

色相: テーマ色相範囲全体

#### アクセントカラー（静的サジェスト）

「強すぎない色が適している」= v, b, dp を除外

| 項目 | サジェスト条件 |
|---|---|
| 色相 | テーマ色相範囲全体 |
| トーン | dkg, dk, g, ltg |

---

### 2-6. ダイナミック (dynamic)

**イメージ**: 強い・はっきりした・派手な
**配色の特徴**: 心理四原色とオレンジ系の高彩度色による色相対比。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | **2, 5, 8, 12, 18**（R, O, Y, G, B の5色相のみ） |
| トーン範囲 | Bk, b, s, v |

#### 色役割の扱い

ダイナミックテーマはベースカラー・アソートカラー・アクセントカラーの主従関係を持たない。
3つの入力UIはすべて同一条件でサジェストを行う。

#### サジェスト（3色共通）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 2, 5, 8, 12, 18（5色相） |
| トーン | Bk, b, s, v |

#### UI仕様

- 面積比率（アクセントカラーあり時）: **1:1:1**（完全均等）

---

### 2-7. ウォームナチュラル (warm-natural)

**イメージ**: 穏やかな・素朴な・温もりのある
**配色の特徴**: 自然環境色（山・田園）の穏やかなトーン。オレンジ系からグリーン系の範囲。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 4〜12（rO〜G の9色相） |
| トーン範囲 | p, ltg, sf, d, dk, dp |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 4〜8（rO〜Y の YR系5色相） |
| トーン | p, ltg, sf, d |

#### アソートカラー（静的サジェスト）

ベースより色相の幅を広げた穏やかな色。

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 4〜12（テーマ色相範囲全体） |
| トーン | ltg, sf, d, dk, dp |

#### アクセントカラー

「アソートカラーと区別する必要はない」= アソートカラーと同一条件

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 4〜12（テーマ色相範囲全体） |
| トーン | ltg, sf, d, dk, dp |

---

### 2-8. フレッシュナチュラル (fresh-natural)

**イメージ**: 若々しい・新鮮な・爽やかな
**配色の特徴**: イエローグリーン〜ブルーグリーン系の明清色中心。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 9〜16（gY〜gB の8色相） |
| トーン範囲 | p, lt, b, W, ltGy |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 9〜16（テーマ色相範囲全体） |
| トーン | p, lt, W, ltGy |

> b トーンはテーマ範囲内だが、ベースカラーとしてはサジェストしない（アクセントカラー向けの選択肢として保留）。

#### アソートカラー（動的サジェスト）

| ベースのトーン種別 | アソートのサジェストトーン |
|---|---|
| 無彩色（W, ltGy） | p, lt |
| 有彩色（p, lt） | W, ltGy |

色相: 9〜16（テーマ色相範囲全体）

#### アクセントカラー

「アソートカラーと区別する必要はない」= アソートカラーと同一条件（動的）

---

### 2-9. モダン (modern)

**イメージ**: 現代的・人工的
**配色の特徴**: 金属・ガラス・コンクリートの都会的なイメージ。無彩色のハイコントラストにブルー系有彩色を組み合わせる。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 16, 17, 18（gB, bB, B の3色相） |
| トーン範囲 | ltg, sf, d, b, s, dp, v, W, ltGy, mGy, dkGy, Bk |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 16〜18（テーマ色相範囲全体）または無彩色 |
| トーン | ltg, ltGy（最適）、次点として mGy |

#### アソートカラー（静的サジェスト）

ハイコントラストの実現を目的とする。

| 優先度 | サジェストトーン | 意図 |
|---|---|---|
| 高 | mGy, dkGy, Bk | 無彩色によるハイコントラスト |
| 中 | sf, d, b, s, dp, v | B系有彩色による色みの追加 |

色相: 16〜18（有彩色を選ぶ場合）または無彩色

#### アクセントカラー（動的サジェスト）

アソートカラーの種別に応じて対になる色を選ぶ。

| アソートのトーン種別 | アクセントのサジェストトーン |
|---|---|
| 無彩色（mGy, dkGy, Bk） | sf, d, b, s, dp, v（B系有彩色） |
| 有彩色（sf, d, b, s, dp, v） | mGy, dkGy, Bk（無彩色系） |

色相: アソートが無彩色の場合は 16〜18、アソートが有彩色の場合は無彩色または 16〜18

---

### 2-10. ロマンチック (romantic)

**イメージ**: かわいい・可憐な・愛らしい
**配色の特徴**: 赤紫〜赤みの黄まで、p・lt トーンのやわらかな配色。ピンク（赤系の p, lt）が特に有効。

#### テーマ制約

| 項目 | 値 |
|---|---|
| 色相範囲 | 24, 1, 2, 3, 4, 5, 6, 7（RP〜rY の8色相） |
| トーン範囲 | p, lt, W, ltGy |

#### ベースカラー（静的サジェスト）

| 項目 | サジェスト条件 |
|---|---|
| 色相 | 24, 1〜7（テーマ色相範囲全体） |
| トーン | p（中心）、W, ltGy |

#### アソートカラー（動的サジェスト）

| ベースのトーン種別 | アソートのサジェスト |
|---|---|
| 有彩色（p） | 色相差 0〜3（同一〜類似）かつ 24,1〜7 の範囲内、トーン p, lt, W, ltGy |
| 無彩色（W, ltGy） | 色相 24,1〜7（テーマ範囲全体）、トーン p, lt |

#### アクセントカラー

「アソートカラーと区別する必要はない」= アソートカラーと同一条件（動的）

## 4. サジェスト計算の実装ガイド

### 4-1. インターフェイス定義（概念）

```typescript
type ThemeId =
  | 'elegant' | 'casual' | 'classic' | 'clear' | 'chic'
  | 'dynamic' | 'warm-natural' | 'fresh-natural' | 'modern' | 'romantic';

type ColorRole = 'base' | 'assort' | 'accent';

interface SelectedColor {
  hueNumber: number | null; // null = 無彩色
  toneSymbol: string;       // トーン記号（例: 'p', 'ltGy', 'Bk'）
}

interface SuggestInput {
  theme: ThemeId;
  role: ColorRole;
  baseColor?: SelectedColor;   // ベースカラーの現在値
  assortColor?: SelectedColor; // アソートカラーの現在値（accent計算時のみ使用）
}

interface SuggestOutput {
  suggestedHues: number[];   // サジェストする色相番号の配列（有彩色のみ）
  suggestedTones: string[];  // サジェストするトーン記号の配列（有彩色・無彩色バケット混在可）
}
```

### 4-2. 計算フロー

```
function computeSuggest(input: SuggestInput): SuggestOutput {
  1. テーマの allowedHues・allowedTones を基底条件として取得
  2. 役割（base / assort / accent）に応じた静的条件を適用
  3. 動的条件（baseColor / assortColor の選択値に依存）があれば追加適用
  4. 絞り込まれた色相・トーンの集合を返す
  5. 結果が空の場合、テーマの allowedHues・allowedTones をそのまま返す（フォールバック）
}
```

### 4-3. 初期値の決定フロー

```
1. computeSuggest({ theme, role: 'base' }) でサジェストを計算
2. サジェスト内からランダムにベースカラーの初期値を選択
3. computeSuggest({ theme, role: 'assort', baseColor: <2の選択値> }) でサジェストを計算
4. サジェスト内からランダムにアソートカラーの初期値を選択
5. アクセントカラーが表示状態の場合:
   computeSuggest({ theme, role: 'accent', baseColor: <2の選択値>, assortColor: <4の選択値> })
   サジェスト内からランダムに初期値を選択
```

### 4-4. 「1段階鮮やかなトーン」のユーティリティ関数（概念）

```typescript
const ONE_STEP_MORE_CHROMATIC: Record<string, string> = {
  p: 'lt',
  ltg: 'sf',
  g: 'd',
  dkg: 'dk',
  lt: 'b',
  sf: 'dp',
  d: 'dp',
  dk: 'dp',
};

function getOneStepMoreChromatic(tone: string): string | null {
  return ONE_STEP_MORE_CHROMATIC[tone] ?? null;
}
```

### 4-5. 色相差の計算（参照）

`color-analysis-rules.md §6` に従い、24色相環上の最短距離（0〜12）で計算する。

```typescript
function hueDistance(a: number, b: number): number {
  const diff = Math.abs(a - b);
  return Math.min(diff, 24 - diff);
}
```

## 5. カラーロール説明文

各テーマの色選択UIで、カラーロールラベルの近くに表示するガイダンステキスト。

### 5-1. エレガント (elegant)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 紫〜赤紫系の淡いパープル（pトーン・ltgトーン） |
| アソートカラー | ベースと同じ色相グループから、ひとつ鮮やかなトーンの色 |
| アクセントカラー | ベース・アソートと調和する、控えめなトーンの色 |

### 5-2. カジュアル (casual)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 橙〜黄系の明るい色（pトーン・ltトーン） |
| アソートカラー | ベースと色相対比が生まれる色（色相差4以上） |
| アクセントカラー | 全体を引き締める高彩度色 |

### 5-3. クラシック (classic)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 茶系の深みある色（dkg・dkトーン） |
| アソートカラー | ベースと同系色の落ち着いたトーン |
| アクセントカラー | 緑〜青系の引き締め色 |

### 5-4. クリア (clear)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 青緑〜青系の澄んだ色、またはホワイト系 |
| アソートカラー | ベースが有彩色ならホワイト系、ホワイト系なら有彩色 |
| アクセントカラー | テーマの清潔感を保つ、控えめなトーンの色 |

### 5-5. シック (chic)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 無彩色（黒・グレー）または渋みのある低彩度色 |
| アソートカラー | ベースが無彩色なら有彩色、有彩色なら無彩色 |
| アクセントカラー | テーマの落ち着きを損なわない渋みある色 |

### 5-6. ダイナミック (dynamic)

ダイナミックテーマでは3色の主従関係がないため、各ロールの説明は選択順を示すガイドとして機能する。

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 赤・橙・黄・緑・青のいずれかの高彩度色または黒 |
| アソートカラー | ベースとは異なる色相の高彩度色 |
| アクセントカラー | さらに別の色相を加えて力強いコントラストを完成させる色 |

### 5-7. ウォームナチュラル (warm-natural)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 橙〜黄系の自然を感じる穏やかな色 |
| アソートカラー | 大地・草木を連想させる自然の色 |
| アクセントカラー | アソートと同じ基準で選ぶ穏やかな色 |

### 5-8. フレッシュナチュラル (fresh-natural)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 黄緑〜青緑系の爽やかな色、またはホワイト系 |
| アソートカラー | ベースが有彩色ならホワイト系、ホワイト系なら有彩色 |
| アクセントカラー | アソートと同じ基準で選ぶ清潔感のある色 |

### 5-9. モダン (modern)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 青系の彩度を抑えた色（ltgトーン）またはライトグレー |
| アソートカラー | 濃いグレー・黒などの無彩色、または青系の有彩色 |
| アクセントカラー | アソートとは反対の種別（無彩色↔有彩色）の色 |

### 5-10. ロマンチック (romantic)

| カラーロール | 説明文 |
|---|---|
| ベースカラー | 赤紫〜橙系のパステルカラー、またはホワイト系 |
| アソートカラー | ベースと近い色相の、やわらかなトーンの色 |
| アクセントカラー | アソートと同じ基準で選ぶかわいらしい色 |
