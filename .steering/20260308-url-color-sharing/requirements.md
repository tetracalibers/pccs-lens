# 要求内容：URLによるカラーコード共有

## 変更・追加する機能の説明

`/approximate` ページで入力したカラーコードを URL のクエリパラメータに反映する。
これにより、特定の色の近似結果ページをブックマーク保存・URL 共有できるようになる。

## ユーザーストーリー

- ユーザーとして、近似結果を出した URL を他の人に送り、同じ色で結果を確認してもらいたい
- ユーザーとして、よく使う色の近似結果ページをブックマークに保存し、すぐに開けるようにしたい

## URL設計

```
/approximate?color=EE0026
```

- パラメータ名: `color`
- 値: `#` を除いた6桁の大文字・小文字を問わない HEX コード（URLエンコード不要）

## 受け入れ条件

1. カラーピッカーまたはテキスト入力で有効な6桁 HEX コードを入力すると、URL が `?color=RRGGBB` 形式に自動更新される
2. `?color=RRGGBB` 付きの URL に直接アクセスすると、そのカラーコードが初期値として復元される
3. URL 更新はブラウザ履歴を汚さない（`replaceState` を使用）
4. テキスト入力の途中（不完全な HEX コード）では URL は変化しない
5. 不正な `color` パラメータが URL に含まれる場合、デフォルト色 `#EE0026` で動作する

## 制約事項

- 変更するファイルは `app/src/routes/approximate/+page.svelte` のみ
- `ColorPicker.svelte` および色計算ロジックは変更しない
- SvelteKit の `$app/stores` および `$app/navigation` を使用する
