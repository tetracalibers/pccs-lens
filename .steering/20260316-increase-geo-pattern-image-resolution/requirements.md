# 要求内容：幾何パターン画像の保存・共有解像度向上

## 変更・追加する機能の説明

幾何パターン（バウハウス風・ジオメトリック）をPNGとして保存・共有する際の画像解像度を、現状の 300×300px から 1200×1200px に引き上げる。

## 背景・動機

- 現状の 300×300px はモバイル端末のRetina/高解像度ディスプレイで拡大すると粗く見える
- Web Share API 経由で SNS（Instagram、LINE など）に送った際にも画質が劣化する
- PNG生成はユーザーがボタンを押したときのみ実行されるため、通常操作（色変更・パターン生成）のパフォーマンスには影響しない
- SVG はベクター形式のため、拡大レンダリングのコスト増加は軽微

## ユーザーストーリー

- ユーザーとして、保存・共有した幾何パターン画像をスマートフォンの画面でフルサイズ表示したとき、ピクセルの粗さを感じることなく鑑賞したい

## 受け入れ条件

- [ ] 保存・共有される PNG が 1200×1200px で生成される
- [ ] 通常操作（色スライダー操作・パターン再生成）の体感速度に変化がない
- [ ] バウハウス風・ジオメトリックの両パターンに適用される

## 制約事項

- 解像度は 1200×1200px（2400 はオーバースペック）
- 変更箇所は `generatePngBlob` 関数内の canvas サイズのみ
