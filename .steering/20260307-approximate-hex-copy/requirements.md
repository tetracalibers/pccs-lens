# 要求内容：近似結果カードのHEXコード表示＋コピーボタン

## 変更・追加する機能の説明

`/approximate` ページのPCCS近似結果カードにおいて、現在表示しているΔE₀₀の値を削除し、代わりにPCCS色のHEXコードとそのコピーボタンを設置する。

## ユーザーストーリー

- ユーザーとして、近似結果の色のHEXコードをひと目で確認したい
- ユーザーとして、HEXコードをワンクリックでクリップボードにコピーしたい

## 受け入れ条件

- [ ] 各近似結果カードにPCCS色のHEXコード（例：`#EE0026`）が表示される
- [ ] HEXコードの隣にコピーボタンが配置されている
- [ ] コピーボタン押下でHEXコードがクリップボードにコピーされる
- [ ] コピー成功後、一時的にフィードバック（例：ボタンの表示変化）が表示される
- [ ] ΔE₀₀の表示は削除される

## 制約事項

- Clipboard API（`navigator.clipboard.writeText`）を使用する
- 既存のUIスタイルと一貫性を保つ
- 外部ライブラリは追加しない
