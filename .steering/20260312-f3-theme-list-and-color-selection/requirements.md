# 要求内容：F3 イメージ別配色シミュレータ（機能1〜4）

## 変更・追加する機能の説明

`product-requirements.md` の F3 要件のうち、以下の1〜4を実装する。

1. `/patterns` にテーマ一覧ページを作成し、任意のテーマページ（`/patterns/[theme]`）に遷移できる
2. テーマページの冒頭に、そのテーマの配色ルール解説文を表示する
3. ベースカラー・アソートカラー・アクセントカラーそれぞれの色相・トーンをインタラクティブに選択できる
   - 色相選択UIと トーン選択UIをセットで提供し、推奨される色相・トーンをサジェストとして視覚的に示す
   - アクセントカラーは任意（初期は非表示・ボタンで追加できる）
4. 選択した色に基づき、面積比率に応じた配色プレビュー（色スウォッチ列）をリアルタイムで更新する
   - 各スウォッチにはPCCS表記（例：`v2`）とHEXコードのコピーボタンを表示する
   - 色相・トーンが変更されたとき、配色プレビューはリアルタイムで更新する

> F3の5〜7（幾何パターン生成・ダウンロード）は別ステアリングで対応する。

---

## ユーザーストーリー

- デザイン学習者として、テーマ一覧からイメージを選び、そのイメージに合う色を配色ルールに沿って選択したい
- 色彩検定の学習者として、テーマの配色ルール解説を読みながら実際に色を選んで体感的に学びたい
- PCCSを使ったデザイナーとして、推奨色相・トーンのサジェストを参考にしながら自分好みの配色を微調整したい

---

## 受け入れ条件

### テーマ一覧（/patterns）

- [ ] 10テーマ（elegant / casual / classic / clear / chic / dynamic / warm-natural / fresh-natural / modern / romantic）の一覧が表示される
- [ ] 各テーマカードにはテーマ名（日本語・英語）とイメージ説明が表示される
- [ ] テーマカードをクリックすると `/patterns/[theme]` に遷移する
- [ ] ナビゲーションバーに「配色シミュレータ」リンクが追加され、`/patterns` に遷移できる

### テーマページ（/patterns/[theme]）

- [ ] ページ上部にテーマの配色ルール解説文が表示される（`image-based-color-rules.md` の各テーマ定義を参照）
- [ ] 存在しないテーマ名の場合は404または一覧ページへリダイレクトする

### 色選択UI

- [ ] ベースカラー・アソートカラーそれぞれに、色相選択UIとトーン選択UIがセットで表示される
- [ ] 「アクセントカラーを追加する」ボタンが表示され、押下でアクセントカラーの入力UIが表示される
- [ ] 色相選択UIは24色相を視覚的に表示し、そのテーマの `allowedHues` に含まれる色相のみをハイライト（サジェスト）表示する
- [ ] トーン選択UIはトーン概念図に基づき、そのテーマの `allowedTones` に含まれるトーンのみをハイライト（サジェスト）表示する
- [ ] `image-based-color-rules.md` の各テーマ定義に従い、役割（base/assort/accent）に応じた動的・静的サジェストを適用する
- [ ] 初期値はサジェスト内からランダムに選択する（決定順：ベース → アソート → アクセント）
- [ ] ページを再表示するたびに初期値がランダムに変わる
- [ ] 無彩色が選択されている場合、色相選択UIには色相ハイライトを表示しない（`image-based-color-rules.md §1-9` に従う）
- [ ] 無彩色選択時に色相扇形をクリックすると有彩色へ切り替わり、トーンが有彩色サジェスト内からランダムに更新される

### 配色プレビュー

- [ ] 選択中のベース・アソート・（任意）アクセントカラーのスウォッチ列が表示される
- [ ] 面積比率に応じた幅（`image-based-color-rules.md §1-6` 参照）でスウォッチが表示される
  - アクセントなし: ベース7：アソート3
  - アクセントあり（通常）: ベース6：アソート3：アクセント1
  - ダイナミック（アクセントあり）: 1：1：1
- [ ] 各スウォッチにはPCCS表記とHEXコードのコピーボタンが表示される
- [ ] 色相・トーンの変更で配色プレビューがリアルタイムに更新される

---

## 制約事項

- PCCSカラーデータは既存の `pccs_colors_full.json` を使用する（全色相を網羅しているため）
- 色の選択はPCCSの hue×tone の組み合わせで管理し、対応する PCCSColor を `pccs_colors_full.json` から取得する
- サジェストは「選択の制限」ではなく「ハイライト」であり、テーマ範囲外の色相・トーンも選択可能
- F3の5〜7（幾何パターン生成・ダウンロード）は本ステアリングのスコープ外
