# 要求内容

## 概要

`ToneImageDiagram.svelte` のツールチップ実装を Popover API に移行し、JSコードを削減する。

## ユーザーストーリー

- 開発者として、ツールチップの開閉制御に関する手動イベント管理コードを減らし、ブラウザネイティブの機能に委ねたい

## 受け入れ条件

- [ ] Escape キーでツールチップが閉じる（ブラウザが自動処理）
- [ ] ツールチップ外クリックで閉じる（light dismiss）
- [ ] セルクリックでツールチップが開く
- [ ] CSS Anchor Positioning による位置制御が維持されている
- [ ] 開閉アニメーションが動作する

## 制約事項

- 「スクロール時に閉じる」要件は意図的に緩める（UX許容済み）
- CSS Anchor Positioning との共存が必要
