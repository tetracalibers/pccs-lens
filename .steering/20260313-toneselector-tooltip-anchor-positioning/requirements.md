# 要求内容：ToneSelector グレー細分ツールチップの左右自動切替

## 背景

`ToneSelector.svelte` のグレーバケットセル（ltGy / mGy / dkGy）をクリックすると表示される
細分選択ツールチップは、常にセルの左側に固定表示されていた。
モバイル端末では ToneSelector が画面左寄りに配置されることが多く、
ツールチップが画面外に見切れてしまう問題があった。

## 変更要求

- CSS Anchor Positioning を用いてツールチップの表示位置を制御する
- デフォルトはセルの左側に表示する
- 左側に収まらない（ビューポートからはみ出す）場合は右側にフォールバックする

## 制約事項

- ページ内に `ToneSelector` が複数配置された場合でも、
  操作したインスタンスのセルを基準にツールチップが表示されること
  （アンカー名の衝突を防ぐこと）
